# WDB-education

This is for project 1 of Web Development Decal @ Berkeley, which is to build a template for a personal website with the work, about, and contact info populated.

`index.html`: file with the main html code. </br>
`template.css`: the style document associated with the html file.  </br>
`about.html`: the html with the introduction section referenced in `index.html`. </br>
`work.html`: the html with the work section referenced in `index.html`. </br>
`peach_cat_shirt.png`: the image included in `index.html`.
